freeCodeCamp

freeCodeCamp is a website where you can learn to code for free. It offers a fantastic range of coursework and projects, whether you’re just getting started in coding or have been doing it for years.

I’ve been coding for quite a while now, but I wanted to learn some of the basics that tend to be skipped over when you’re working to a deadline, cutting and pasting code from Stackoverflow without fully understanding it… did I say that out loud?

There’s something satisfying about learning a technology from scratch, understanding the core building blocks.

Responsive web design certification

An introductory course on HTML, CSS, applied visual design and accessibility. If you’ve never built a web page before this is a great starting point.

I found the curriculum fairly straightforward but learned lots also. CSS tricks I had no idea about, accessibility fundamentals, and HTML tag/structure best practice.
